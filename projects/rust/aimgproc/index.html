AImgProc

This was a project I made in prep school for the TIPE (which is a project presented for the last exam). It is a small tool that allows to script an OpenCL pipeline using the rhai scripting language. It was made to quickly apply some image processing in bulk before feeding the images to a neural network (to help with image recognition). Sadly this project does not work anymore because of an error in a dependency of the OpenCL library (ocl) I use - it does not compile -_-.

The source code is available on github under the MIT license.

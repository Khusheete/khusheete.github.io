Lambda Calculus Implementation

This is an interpreter/repl for a custom λ-calculus dialect I made to play around with this very interesting bit of mathematics and computer science.

It implements both "lazy" and "strict" β-reduction (this is akin to strict and lazy evaluation). Strict β-reduction is the usual way programs are executed in most programing languages, and allows to create most programs in this λ-calculus dialect. However for and while loops require some use of the Y combinator which leads to an infinite loop when using strict β-reduction. This is why there is a need for lazy β-reduction. It allows the implementation of general purpuse loops that can be used afterwards.

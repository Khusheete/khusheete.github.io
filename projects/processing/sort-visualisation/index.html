Sorting Algorithm Visualisation

This is the classic visualisation of sorting algorithms. Implemented algorithms are:

  • Bubble sort
  • Cocktail sort
  • Comb sort
  • Gnome sort (or Stupid sort)
  • Insertion sort
  • Odd-Even sort
  • Quicksort
  • Selection sort
  • Shellsort

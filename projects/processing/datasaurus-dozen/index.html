Datasaurus Dozen

This project is based on this paper which shows that statistical properties of a dataset (eg. mean, standard deviation, correlation...) are not enough to fully understand it: given a dataset with some statistics, it is possible to create an other dataset with the same statistics (down to some wanted precision) but with a drasticly different shape (eg. a dinosaur !).

To do that, we nudge datapoints randomly, but only if it keeps the same statistics and if the overall distance from the dataset to the desired shape decreases.

Elemental Words

Have you ever dreamt you could write words using only elemental symbols ? No ? Well here is a program that automatically does that anyway. Launch it in an interactive python shell, and use the get_coverage function with a word, and it will be transformed as such: SUPErB SeNTeNCe ThAt WAs PaIn (<- very hard to have a writable sentence).

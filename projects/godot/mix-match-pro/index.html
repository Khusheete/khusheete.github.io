Mix Match Pro

By Higgaion Games

This is a classical match-3 game in which you have a limited amount of turn to get the highest score.

We made this game to experiment with working with version control systems in a medium sized team. We also decided to port it to android to learn how to develop for this platform.


The game is available to play on itch.io. There are downloads for windows, linux, and android.
